数组

基础

 数组元素是任意类型的

 最大索引（2^32-2）

创建

 直接量

  var arr=[1,1,1]; 省略的值为undefined； [ , , ]长度为2

 构造函数Array（）

  new Array（）空数组 等于 [ ];

  new Array(10) 指定长度，注意此时元素的索引尚未定义

  new Array(item1,item2)

 稀疏数组

  length>元素个数

 length属性

  当设置length小于数组当前长度时，索引值大于等于length的元素被删除

基本方法

 添加

  索引赋值

  push( ) 末尾添加，返回新长度

  unshift( )首部添加，返回新长度

 删除

  delete 删除元素，不会改变length长度

  pop( )末尾删除，返回删除的元素

  shift( )首部删除，返回删除的元素

 join( )：把数组中所有元素转换成字符串连接在一起返回，可以指定一个间隔符

 reverse( ) :数组逆序，改变原数组

 sotr( )

  不传参数按字母表排序

  传入比较函数：function(a,b) {} 如果a在前 返回小于0的数值

 concat( ):数组合并 返回新数组

 slice( start,end): 数组截取包含start不包含end，不改变原数组

 spice( start,count) 从起始位置删除count个元素，返回删除的元素，改变原数组

 栈方法(LIFO)

  push() 入栈

  pop() 出栈

 队列方法(FIFO)

  push()入队

  shift() 出队

 反队列方法(FILO)

  unshift()前端入队

  pop() 尾端出队

 toString() 把每个元素装换为字符串，输出用逗号分隔的字符串列表

 toLocaleString()返回本地字符串表示

高级方法

 arr.forEach(function(item,index,arr){ }) 会改变原数组

 var anotherArr=arr.map(function(item,index,arr){ return }) 必须有return，不会改变原有数组

 filter(),返回经过过滤后为true的元素数组，可以用来过滤稀疏数组

 every()全为true时返回true和some()至少有一个为true时返回true

 化简操作

  reduce()和reduceRight()

 indexOf(item,start)和lastIndexOf(item,start) 数组元素搜索

 检测数组

  Array.isArray()

  instanceof 存在frame的问题
